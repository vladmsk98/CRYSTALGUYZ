# Мы любим Московский транспорт — Social Media Card

Минималистичная веб-визитка для проекта **"Мы любим Московский транспорт"**.  
Содержит баннер и ссылки на официальные сообщества в социальных сетях: **ВКонтакте** и **Дзен**.  
Создана только на HTML + CSS.

---

## 🎯 Цель
- Предоставить **чистую и доступную** визитку проекта.
- Укрепить **визуальный образ** с помощью изображения-баннера.
- Обеспечить **лёгкий доступ** к социальным сетям проекта.
- Сохранить **минимализм** и **доступность**.

---

## ✨ Особенности
- **Изображение-баннер** (`BANNER.jpg`) — размещено в начале карточки, центрировано, с тенями и эффектами, соответствующими дизайну.
- **Ссылки на соцсети** — оформлены как интерактивные кнопки с цветами, соответствующими платформам.
- **Чистая семантика** — доступность, скринридеры, SEO.
- **Адаптивный дизайн** — корректное отображение на мобильных и десктопах.

---

## 🛠 Технологии
- **HTML5** — семантическая разметка (`<main>`, `<img>`, `<a>`)
- **CSS3** — переменные, `backdrop-filter`, `box-shadow`, адаптивные медиа-запросы
- **Шрифт**: Inter (system fallback — Segoe UI, sans-serif)
- **Без JS**, без внешних зависимостей

## 🖼 Особенности дизайна
| Элемент                    | Реализация |
|---------------------------|------------|
| Фон                        | Глубокий `#0d1b2a` — тёмно-синий |
| Кард                       | `rgba(18,30,45,0.85)` + `backdrop-filter: blur(10px)` |
